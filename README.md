# Сайт-візитівка: Святкова майстерня

Базовий стартовий шаблон для односторінкового сайту-візитівки.

## Структура

- `index.html` — основна сторінка
- `portfolio.html` — окрема сторінка з виробами
- `assets/styles/main.css` — стилі, адаптивність, базові анімації
- `assets/scripts/main.js` — анімації при скролі, плавна навігація, легкий parallax

## Запуск

Відкрийте `index.html` у браузері або запустіть локальний сервер (наприклад, через Live Server у VS Code).
